Key Features

Ant-Cache provides a complete caching solution with excellent performance and clean API design

Multiple Data Types

Support for String, Array, and Object data types with flexible storage options.

TTL Support

Flexible expiration with time units (s/m/h/d/y) for automatic cache invalidation.

Atomic Operations

SETNX family commands for concurrency safety and thread-safe operations.

Data Persistence

ATD snapshots + ACL command logs for reliable data recovery and backup.

High Performance

1M+ operations per second with optimized algorithms and data structures.

Simple Deployment

Single binary with no dependencies, easy configuration and deployment.
